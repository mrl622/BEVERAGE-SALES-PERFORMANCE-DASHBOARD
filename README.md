Beverage Sales Performance Dashboard 🥤
Power BI Data Analytics Project
📌 Project Overview
This project focuses on analyzing sales data for a beverage distributor across multiple brands and regions. The goal was to transform raw transactional data into a high-level executive dashboard to track Revenue, Profitability, and Market Share.
By visualizing 16M units sold, this dashboard identifies which brands drive the highest margins and which regions require strategic intervention.
📊 Key Metrics & Logic (Double-Checked)
Total Revenue: $8.17M
Total Units Sold: 16,000,000
Total Profit: $6.41M
Operating Profit Margin: 78.54% (Calculated as \frac{\text{Total Profit}}{\text{Total Revenue}})
Expense Analysis: Total expenses of $1.75M tracked against revenue to ensure sustainable growth.
🛠️ Technical Stack
Tool: Power BI Desktop
Data Transformation: Power Query (M) for data cleaning, type conversion, and handling null values.
Data Modeling: Star Schema design for optimized report performance.
Calculations: Custom DAX (Data Analysis Expressions) for time-intelligence and margin calculations.
🔍 Key Insights
Brand Dominance: Coca-Cola and Dasani represent the largest share of revenue, with Coca-Cola leading in total volume.
Regional Performance: Revenue is heavily concentrated in specific regions (e.g., East and West), suggesting a potential for expansion in under-penetrated markets.
Profitability: The portfolio maintains an exceptionally high profit margin of ~78%, indicating low COGS (Cost of Goods Sold) or premium pricing strategies.
